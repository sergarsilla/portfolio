# Easter Eggs - Portfolio

Easter eggs interactivos escondidos en el portfolio.

---

## Lista de Easter Eggs

### 1. Konami Code → Matrix Mode
- **Activación:** Presiona `↑ ↑ ↓ ↓ ← → ← → B A` (código Konami)
- **Efecto:** Cambia los colores a magenta, confetti, alerta "Access Level: ROOT Granted"
- **Deshabilitar:** Repite la secuencia para desactivar

### 2. Ctrl+Shift+T → Secret Terminal
- **Activación:** Presiona `Ctrl + Shift + T`
- **Efecto:** Abre una terminal flotante
- **Cerrar:** Presiona `Escape` o escribe `exit`

### 3. Logo Click → Debug Mode
- **Activación:** Haz clic en el logo del header 5 veces rápidamente
- **Efecto:** Muestra panel de debug (FPS, memoria, tiempo de render)
- **Cerrar:** Presiona `Escape`
