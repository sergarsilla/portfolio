# 🎮 Easter Eggs Guide - Portfolio de Sergio García Mansilla

## 🕹️ Cómo Activar los Easter Eggs

### 1. Terminal Oculta 💻

**Activación**: Presiona `Ctrl + Shift + K`

**Descripción**: Se abrirá una terminal interactiva con comandos reales.

**Comandos Disponibles**:

```bash
# Mostrar ayuda
help

# Información del usuario
whoami

# Listar proyectos
ls

# Ver enlaces del CV
cat cv

# Árbol de habilidades
skills

# Información de contacto
contact

# Desafío CTF
hack

# Activar modo Matrix
matrix

# Limpiar terminal
clear

# Cerrar terminal
exit
```

**CTF Challenge**: 
- Ejecuta el comando `hack` para ver el desafío
- Decodifica el mensaje Base64: `U2VyZ2lvIGxvdmVzIGN5YmVyc2VjdXJpdHkh`
- Usa el comando `decode <tu_respuesta>` para verificar
- Pista: Es un método de codificación común

**Solución**: 
```bash
decode sergio loves cybersecurity!
```

---

### 2. Konami Code 🎮

**Activación**: Presiona la secuencia: `↑ ↑ ↓ ↓ ← → ← → B A`

**Efecto**: 
- Activa el "Modo Hacker"
- Cambia toda la interfaz a tema Matrix (verde)
- Muestra mensaje "Access Granted - Welcome, Neo"
- Efecto de lluvia Matrix (opcional)

**Cómo hacerlo**:
1. Asegúrate de que el foco esté en la página (haz clic en cualquier parte)
2. Usa las flechas del teclado: Arriba, Arriba, Abajo, Abajo, Izquierda, Derecha, Izquierda, Derecha
3. Luego presiona las teclas `B` y `A`

## 🐛 Troubleshooting

**La terminal no se abre**:
- Asegúrate de presionar `Ctrl + Shift + K` correctamente
- Verifica que el foco esté en la página
- Intenta hacer clic en la página primero

**El Konami Code no funciona**:
- Usa las flechas del teclado, no el numpad
- Presiona las teclas en secuencia sin pausas largas
- Asegúrate de terminar con `B` y `A` (letras)

**El modo Matrix no se activa**:
- Completa el Konami Code correctamente
- O usa el comando `matrix` en la terminal
- Recarga la página si es necesario
