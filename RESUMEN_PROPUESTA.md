## 🎮 Guía Rápida de Easter Eggs

### Para Probar Ahora:

1. **Abre la terminal**: `Ctrl + Shift + K`
   - Escribe `help` para ver comandos
   - Prueba `whoami`, `ls`, `skills`
   - Intenta el CTF: `hack`

2. **Activa modo Matrix**: `↑↑↓↓←→←→BA`
   - Usa las flechas del teclado
   - Termina con B y A

3. **Explora el código fuente**: `F12`
   - Busca el ASCII art
   - Lee los mensajes ocultos

4. **Usa la navegación lateral**:
   - Solo visible en desktop
   - Click en iconos para navegar
   - Observa el indicador activo
